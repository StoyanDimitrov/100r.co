introduction

Nasu is a spritesheet editor created to help us design and assemble the assets of our famicom games. It can import and export both .chr spritesheets. If you wish to learn more about creating 8bit projects, see our Assembly Guide.

Meet Nasubit, the mascot for Nasu. Nasubit is there to talk about updates relating to Nasu.

sprite editor

The viewport is divided into 3 regions, the Sprite Editor, the Tile Selector and the Nametable Editor. The Sprite Editor allows you to see 16 tiles at once, and paint sprites in 4 colors. You can navigate the sprite editor with the wasd keys, and you can change the brush color with the number keys(1 to 4).

The application was initially built to help with porting of Donsol to the Famicom, and later evolved to include the features of the drawing software Noodle. It is currently used to create both Famicom projects such as spacetime, and various assets for Dotgrid and Orca.

desktop version

We create our own tools, built light to reduce our energy use, also to lessen our dependence on closed-source applications we can't repair ourselves. Nasu is available as standalone application written in portable C89. It is distributed as a single file C file that build into a 20kb executable designed to runs offline and having only SDL2 as dependency.
